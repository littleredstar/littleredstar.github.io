2017.3.15模拟赛T1

By Black_Moon

发表于 2017-03-15

Description 

求N位的由‘0’、‘1’、‘2’组成的所有前缀都满足 num2(‘2’的个数)≥num1≥num0 的字符串个数.

Solution

很水的DP题,设f[i][j][k]表示当前i位,num1=j,num0=k时的方案数(第一维可以用滚动数组压掉).显然f[i][j][k]=f[i-1][j][k]+f[i-1][j-1][k]+f[i-1][j][k-1]，边界特判一下就好了.
