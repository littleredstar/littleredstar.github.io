BZOJ1007: [HNOI2008]水平可见直线

By Black_Moon

发表于 2017-03-29

传送门：http://www.lydsy.com/JudgeOnline/problem.php?id=1007

Solution

裸单调栈。。
就是先按斜率排序，每次进栈判断是否将栈顶覆盖即可，判断是否覆盖可以根据两条直线与栈中直线交点的x坐标大小即可。
