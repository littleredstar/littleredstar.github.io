BZOJ3524/2223: [Poi2014]Couriers

By Black_Moon

发表于 2017-03-20

Description 

给一个长度为n的序列a。1≤a[i]≤n。m组询问，每次询问一个区间[l,r]，是否存在一个数在[l,r]中出现的次数大于(r-l+1)/2。
如果存在，输出这个数，否则输出0。

Input 

第一行两个数n，m。第二行n个数，a[i]。接下来m行，每行两个数l,r，表示询问[l,r]这个区间。

Output 

m行，每行对应一个答案。

Sample Input 

7 5
1 1 3 2 3 4 3
1 3
1 4
3 7
1 7
6 6

Sample Output 

1
0
3
0
4

Hint 

[数据范围]
n,m≤500000

Solution

第一次打主席树模板.好像就是把线段树的儿子连来连去.建n棵线段树,记录每个数出现次数,区间出现次数就像前缀和减一下就可以求出然后二分一下答案就好了.复杂度O(n log n).
